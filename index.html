📦 安装使用

在你的 SolidJS 项目中安装：

npm install solid-web-components-ui

使用示例：

import { Button } from 'solid-web-components-ui'; function App() { return ( <Button variant="primary" size="medium"> 点击我 </Button> ); }

自定义 Token（覆盖 CSS 变量）

在你的项目 CSS 中覆盖变量即可实现主题定制：

:root { --color-primary: #7c3aed; --color-primary-hover: #6d28d9; --color-secondary: #db2777; --spacing-md: 1.25rem; --shadow-md: 0 6px 12px -2px rgba(124, 58, 237, 0.2); }

🔧 开发命令

  • npm run dev - 启动开发服务器
  • npm run build - 构建项目（构建日志会提示缺失 Token）
  • npm run test - 运行测试用例
  • npm run lint - 代码质量检查

📁 项目结构

solid-web-components-ui/ ├── src/ │ ├── components/ │ │ ├── Button.tsx # 按钮组件（使用 CSS 变量） │ │ ├── Button.css # 按钮样式 │ │ └── __tests__/ │ ├── tokens/ │ │ └── utils.ts # Token 工具函数 │ ├── tokens.d.ts # 虚拟模块类型声明 │ └── index.ts # 入口文件 ├── tokens.config.ts # 设计 Token 配置文件 ├── vite-plugin-tokens.ts # Token 生成 Vite 插件 └── vite.config.ts # Vite 配置

ℹ️ 修改 tokens.config.ts 后运行 build，构建日志会自动打印生成的 CSS 变量和缺失 Token 警告。
